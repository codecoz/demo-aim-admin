## AimAdmin Demo Application

A Demo application of AimAdmin. 

## Requirements
- ** PHP 8.2.0 or higher;
- ** PDO-SQLite PHP extension enabled;

### Clone this project & follow the steps

```bash
git clone git@github.com:codecoz/aim-admin.git
```
```shell
composer install
```
```shell
php artisan key:generate
```
```shell
php artisan migrate
```
```shell
npm i
```
```shell
npm run build
```
